# Shadowban-bachelorarbeit

dieses Projekt analysiert Shadow Banning auf Youtube : die verdeckte Reduzierung der Sichtbarkeit von Inhalten ohne dem Content Creator zu benachrichtigen dabei wird untersucht , ob die politische Ausrichtung eines Kanals die Sichtbarkeit in den Suchergebnissen beeinflusst,und ob sich Einschränkungen auf die Performance von Videos auswirken. Zusätzlich wird analysiert: Ob die Größe der Abonnenten eine Rolle dabei spielt, ob politisch extreme Inhalte in ihrer Sichtbarkeit eingeschränkt werden. In welchem Ausmaß sich dies auf die Reichweite auswirkt. Ob die thematische Breite eines Kanals (Spezialisierung oder breites Themenangebot) mit der Wahrscheinlichkeit von Einschränkungen sowie deren Auswirkungen auf die Videoperformance zusammenhängt.

Da der vollständige Datensatz über 2 GB umfasst und nicht sinnvoll in ein Repository hochgeladen werden kann, liegt diesem Projekt eine kleinere Beispieldatei bei.

Forschungsfragen: H1a Politisch extreme Kanäle werden häufiger in den Suchergebnissen eingeschränkt.

H1b Extreme politische Ausrichtung schwächt die negativen Effekte eines Shadowbans auf die Video‑Performance.

H2a Große Kanäle mit extremen politischen Inhalten sind stärker von Einschränkungen betroffen als kleine Kanäle.

H2b Große Kanäle verlieren bei einer Einschränkung weniger Performance als kleine Kanäle.

H3a Thematisch breit aufgestellte Kanäle werden für extreme politische Inhalte seltener eingeschränkt.

H3b Thematisch vielseitige Kanäle verkraften Einschränkungen besser als spezialisierte Kanäle.
